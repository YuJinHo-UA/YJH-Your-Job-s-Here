# Roadmap

## Q1
- API contract hardening with full OpenAPI spec.
- Add DB migrations instead of inline schema bootstrap.
- Introduce role-based endpoint middleware for all APIs.

## Q2
- Notification center UI for mentions, watcher updates, failed runs.
- Advanced reports: module heatmap and MTTR trend.
- Better attachment management with previews.

## Q3
- Team-level permissions matrix.
- Public read-only dashboards with tokenized links.
- CI-ready automated test suite (PHPUnit + API tests).

## Q4
- Optional PostgreSQL backend adapter.
- Background job queue for webhooks and heavy analytics.
- SSO integration (OIDC).
