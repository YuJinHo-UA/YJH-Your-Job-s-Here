# Features

## Bug Tracking
- Full bug lifecycle and priorities/severity.
- Comments, mentions, watchers, history.
- Duplicate detection helpers.
- Saved filters and preview popups.

## Test Management
- Hierarchical plans and suites.
- Rich test cases with steps/checklists.
- Test runs and executions with statuses.
- Create bug directly from failed execution.

## Knowledge Base
- Wiki pages with version history.
- Page compare and rollback-like auditing.

## Kanban
- Boards, columns with WIP limits.
- Drag-and-drop card movement via API.
- Card linking to bug/test/wiki entities.

## Analytics and Risk
- Pass/Fail, trend, high-priority bug charts.
- Release risk score based on bug mix and pass rate.
- Click-through charts to filtered pages.

## UX and Productivity
- Global search (`Ctrl+K`).
- Draft autosave in forms.
- Theme toggle and language switch.
- My Day overview and quick actions.
