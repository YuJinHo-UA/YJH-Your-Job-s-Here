# User Guide

## Login
1. Open `login.php`.
2. Enter credentials.
3. Use profile controls in header to change theme and language.

## Work with Bugs
1. Go to `Bugs`.
2. Use filters (project/status/priority/assignee).
3. Open or create bug.
4. Fill title, description, severity, priority, due date.
5. Add comments and mention teammates using `@username`.

## Test Workflow
1. Create plan in `Test Plans`.
2. Add suites and test cases.
3. Create run in `Test Runs`.
4. Execute each case in `testrun.php`.
5. For failed execution, use `Create bug`.

## Wiki
1. Create page in `Wiki`.
2. Edit and save versions.
3. Compare versions in `wiki-page.php`.

## Kanban
1. Open `Kanban`.
2. Create card.
3. Drag card between columns.
4. Use links to related bugs/tests/wiki.

## Calendar
1. Open `Calendar`.
2. Add absence with type and date range.
