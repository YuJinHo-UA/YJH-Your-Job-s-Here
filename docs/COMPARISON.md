# Comparison

## YJH vs Typical Toolchain

| Capability | YJH | Jira | TestRail | Confluence |
|---|---|---|---|---|
| Bug tracking | Yes | Yes | Limited | No |
| Test plans/cases/runs | Yes | Plugin-heavy | Yes | No |
| Wiki with history | Yes | Basic docs plugin | No | Yes |
| Kanban | Yes | Yes | No | No |
| Risk Engine dashboard | Yes | Custom setup | No | No |
| Local lightweight install | Yes | Heavy | Medium | Heavy |
| Native test->bug flow | Yes | Partial | Partial | No |
| Single DB/file deploy | Yes (SQLite) | No | No | No |

## Positioning
YJH is strongest as:
- Unified QA workspace for small to mid engineering teams.
- Portfolio-grade demonstration of product thinking + full-stack delivery.
- Fast pilot platform for teams that need integrated quality operations without enterprise overhead.
