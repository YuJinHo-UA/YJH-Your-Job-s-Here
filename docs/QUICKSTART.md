# Quick Start (2 minutes)

## 1. Requirements
- PHP 8.1+ with SQLite extension enabled.
- Windows, macOS, or Linux terminal.

## 2. Run
From project root:

```bash
php -S 127.0.0.1:8000
```

Open:
- `http://127.0.0.1:8000/login.php`

Demo credentials:
- `admin@yujin.ho / admin123`
- `qa@yujin.ho / qa123`

## 3. First Tour
1. Open `Dashboard`.
2. Create a bug in `Bugs -> Create Bug`.
3. Create a test run in `Test Runs`.
4. Execute tests in `testrun.php` and create bug from failed case.
5. Open `Wiki` and create a page.

## 4. Data and Files
- DB file: `database.sqlite`
- Uploads: `uploads/`
